# gamblinginator3000 ~ a silly portputer ysws submission :3

hello and welcome to my absoluteely insane submission, in this manic episode you will experience the world's peakiest game written in python with tkinter. Because my girlfriend loves coinflips to decide on stuff and to ragebait me i decided to make a game around that. After i realised it was wayyyy too boring i added slots  , so yah, peak gambling simulator.

I made it in tkinter which was hell but it was fun learning, then i manually made all icons in code at 3 am for ***optimization***, am I insane now? probably, would i have done it again? ofc.

## how to run :3
```bash
git clone https://github.com/Energyboy-OwO/gamblinginator3000.git
cd gamblinginator3000
python main.py
```

and have fun :p
